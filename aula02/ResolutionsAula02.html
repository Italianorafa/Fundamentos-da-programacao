Esta página apresenta uma resolução comentada do exercício Aula02-3.

Problema

Escreva um programa que dados três valores, imprime o maior deles. Consegue resolver só com duas comparações?

Análise

Temos três valores dados, a que podemos chamar n1, n2, e n3. Pretendemos um resultado, a que podemos chamar mx. Queremos
\( mx=n1 \)   se   \( n1 \geq n2 \wedge n1 \geq n3 \),
\( mx=n2 \)   se   \( n2 \geq n1 \wedge n2 \geq n3 \),
\( mx=n3 \)   se   \( n3 \geq n1 \wedge n3 \geq n2 \).

Uma tradução direta para Python, com três instruções if, funciona, mas requer 6 comparações.

Facilmente se percebe que se a primeira condição for falsa, então o máximo só pode ser n2 ou n3, por isso podemos usar if-elif-else e reduzir a 3 comparações.

if n1 >= n2 and n1 >= n3:
    mx = n1
elif n2 >= n3:
    mx = n2
else:
    mx = n3

No entanto, a forma mais económica de resolver resulta do seguinte raciocínio:

  1. Assim que obtemos o primeiro valor, podemos dizer que até agora esse é o máximo, ou seja: \(mx = n1\).
  2. Quando recebemos o segundo valor, esse passa a ser novo máximo se e só se \(n2 > mx\).
  3. Quando recebemos o terceiro valor, fazemos o mesmo.

Ou seja,

mx = n1
if n2 > mx:
    mx = n2
if n3 > mx:
    mx = n3

Este algoritmo é simples e requer menos comparações. Além disso, é muito fácil de generalizar para mais números: cada número extra só vai requerer mais uma comparação e uma atribuição.

Solução

Juntando as intruções para input/output, obtemos:

# max3.py
n1 = float(input("n1? "))
n2 = float(input("n2? "))
n3 = float(input("n3? "))

mx = n1
if n2 > mx:
    mx = n2
if n3 > mx:
    mx = n3

print(mx)

Se intercalarmos os inputs com os ifs, até podemos reutilizar uma única variável para os valores introduzidos:

# max3.py
num = float(input("n1? "))
mx = num                     # mx is the maximum so far

num = float(input("n2? "))
if num > mx:
    mx = num                 # update mx only if latest num is larger

num = float(input("n3? "))
if num > mx:
    mx = num                 # update mx only if latest num is larger

print(mx)
